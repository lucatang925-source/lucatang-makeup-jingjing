## 三、替换图片链接步骤

### 3.1 打开配置文件

1. 找到 `photographer-h5/index.html` 文件
2. 用记事本或文本编辑器打开
3. 找到【全局配置区】

### 3.2 替换头像图片

找到以下代码：
```javascript
avatar: "https://images.unsplash.com/photo-...",
```

替换为：
```javascript
avatar: "https://你的图床链接/头像.jpg",
```

**建议**：使用 image1 作为头像（化妆师静静的形象照）

### 3.3 替换婚纱妆造图片

找到以下代码（约第85-96行）：
```javascript
wedding: [
    "https://images.unsplash.com/photo-...",
    "https://images.unsplash.com/photo-...",
    // ...更多图片
],
```

将示例链接替换为实际上传的图片链接：
```javascript
wedding: [
    "https://sm.ms/image/image3.jpeg",
    "https://sm.ms/image/image4.jpeg",
    // ...继续添加 image5 - image24
],
```

### 3.4 替换秀禾妆造图片

找到以下代码（约第97-108行）：
```javascript
xiuhu: [
    "https://images.unsplash.com/photo-...",
    "https://images.unsplash.com/photo-...",
    // ...更多图片
],
```

将示例链接替换为实际上传的图片链接：
```javascript
xiuhu: [
    "https://sm.ms/image/image25.jpeg",
    "https://sm.ms/image/image26.jpeg",
    // ...继续添加 image27 - image38
],
```

## 四、批量上传技巧

### 4.1 使用 SM.MS 批量上传

1. 打开 https://sm.ms
2. 点击 **"Select image(s)"**
3. 按住 `Ctrl` 键选择多张图片（最多10张）
4. 等待全部上传完成
5. 逐个复制图片链接

### 4.2 使用 PicGo 工具（高级用户）

如果需要频繁上传图片，推荐使用 PicGo：
1. 下载地址：https://molunerfinn.com/PicGo/
2. 配置 SM.MS 图床
3. 拖拽图片即可自动上传并复制链接

## 五、图片链接格式要求

### 5.1 正确的链接格式

```
https://s2.loli.net/2024/01/15/xxxxx.jpg
```

### 5.2 错误的链接格式

```
// 不要带 Markdown 标记
![image](https://...)

// 不要带 HTML 标签
<img src="https://...">

// 不要带其他参数
https://...?w=600&h=800
```

## 六、验证图片是否生效

### 6.1 本地测试

1. 双击 `index.html` 用浏览器打开
2. 检查图片是否正常显示
3. 点击分类标签切换，检查两类图片

### 6.2 常见问题

**问题1：图片显示不出来**
- 检查链接是否完整（以 http:// 或 https:// 开头）
- 检查链接是否能在浏览器直接打开
- 检查是否有多余的空格或换行

**问题2：图片加载慢**
- 建议使用国内图床（SM.MS、路过图床）
- 压缩图片大小（建议单张不超过 2MB）

**问题3：图片顺序不对**
- 调整代码中图片链接的顺序
- 排在前面的图片会先显示

## 七、图片优化建议

### 7.1 图片尺寸
- 建议宽度：800-1200 像素
- 建议格式：JPG（照片）或 PNG（透明图）
- 建议大小：单张不超过 2MB

### 7.2 图片数量
- 每类建议 10-20 张
- 总数不超过 50 张
- 精选优质作品展示

### 7.3 图片选择
- 选择清晰、光线好的照片
- 展示不同风格和造型
- 包含特写和全身照
