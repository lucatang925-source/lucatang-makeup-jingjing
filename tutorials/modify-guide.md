# 小白内容修改教程

> 本教程教你如何在零代码基础的情况下，修改H5网页的内容。只需修改配置文件，无需理解代码。

---

## 一、准备工作

### 1.1 你需要什么工具

**Windows用户：**
- 记事本（系统自带）或 Notepad++（推荐）
- 下载地址：https://notepad-plus-plus.org/downloads/

**Mac用户：**
- 文本编辑（系统自带）或 VS Code（推荐）
- 下载地址：https://code.visualstudio.com/

### 1.2 如何打开文件

1. 找到 `index.html` 文件
2. **不要双击打开**（那样会用浏览器打开）
3. 右键点击文件 → **"打开方式"** → 选择记事本/Notepad++/文本编辑

---

## 二、找到配置区域

### 2.1 配置区在哪里

打开 `index.html` 后，找到标记为 **【全局配置区】** 的部分。

大概在第20行到第120行之间，你会看到：
```javascript
window.PAGE_CONFIG = {
    // 这里是各种配置...
};
```

**重要提示：**
- 只修改这个区域内的内容
- 不要修改区域外的代码
- 不懂的地方不要动

### 2.2 配置区的结构

```javascript
window.PAGE_CONFIG = {
    pageTitle: "...",        // 页面标题
    profile: { ... },        // 个人名片信息
    portfolio: [ ... ],      // 作品集图片
    services: { ... },       // 服务介绍
    contact: { ... },        // 联系方式
    copyright: { ... },      // 版权信息
    theme: { ... }           // 主题颜色
};
```

## 三、修改页面标题

### 3.1 找到位置

在配置区找到：
```javascript
pageTitle: "摄影师小林 | 专业人像摄影",
```

### 3.2 如何修改

把引号里的内容改成你想要的标题：
```javascript
pageTitle: "化妆师小美 | 新娘跟妆专家",
```

**注意事项：**
- 保留两边的引号 `"`
- 标题会显示在浏览器标签页
- 建议控制在20字以内

## 四、修改个人名片信息

### 4.1 找到位置

在配置区找到 `profile` 部分：
```javascript
profile: {
    avatar: "...",
    name: "...",
    tags: [...],
    city: "...",
    intro: "..."
},
```

### 4.2 修改头像

找到：
```javascript
avatar: "https://images.unsplash.com/photo-...",
```

**如何获取图片链接：**
1. 把你的头像上传到图床（推荐：SM.MS https://sm.ms）
2. 上传后复制图片链接
3. 替换引号里的内容

**示例：**
```javascript
avatar: "https://你的图床链接/头像.jpg",
```

**头像建议：**
- 正方形图片最佳
- 尺寸建议 400x400 像素以上
- 文件大小建议不超过 500KB

### 4.3 修改姓名

找到：
```javascript
name: "林晓薇",
```

改成你的姓名：
```javascript
name: "王小美",
```

### 4.4 修改职业标签

找到：
```javascript
tags: ["人像摄影师", "婚礼跟拍", "商业摄影"],
```

**如何修改：**
- 每个标签用引号包起来
- 多个标签用逗号分隔
- 最多建议3-4个标签

**示例：**
```javascript
tags: ["化妆师", "新娘跟妆", "舞台妆造"],
```

### 4.5 修改所在城市

找到：
```javascript
city: "上海",
```

改成你的城市：
```javascript
city: "北京",
```

### 4.6 修改个人简介

找到：
```javascript
intro: "用镜头记录每一个珍贵瞬间，让美好永恒定格",
```

改成你的简介：
```javascript
intro: "专注新娘妆容设计，让每一位新娘都成为最美的新娘",
```

**简介建议：**
- 控制在30字以内
- 突出你的特色和优势
- 避免使用特殊符号

## 五、修改作品集图片

### 5.1 找到位置

在配置区找到 `portfolio` 部分：
```javascript
portfolio: [
    "https://images.unsplash.com/...",
    "https://images.unsplash.com/...",
    // ...更多图片
],
```

### 5.2 如何添加/删除图片

**添加图片：**
1. 把你的作品上传到图床
2. 复制图片链接
3. 在方括号 `[]` 内添加新行：

```javascript
portfolio: [
    "https://原来的链接1",
    "https://原来的链接2",
    "https://新添加的链接",  // 新增的图片
],
```

**删除图片：**
直接删除对应的行即可。

**注意：**
- 每个链接后面要有逗号 `,`
- 最后一个链接后面**不要**逗号

### 5.3 图片要求

- 格式：JPG 或 PNG
- 宽度：建议 800 像素以上
- 数量：最多50张
- 大小：单张建议不超过 2MB

### 5.4 推荐图床

| 图床 | 网址 | 特点 |
|------|------|------|
| SM.MS | https://sm.ms | 免费、稳定、国内可用 |
| 路过图床 | https://imgse.com | 免费、无需注册 |
| 阿里云OSS | https://www.aliyun.com/product/oss | 付费、最稳定 |

## 六、修改服务介绍

### 6.1 找到位置

在配置区找到 `services` 部分。

### 6.2 修改服务内容标题

找到：
```javascript
title: "服务内容",
```

改成你想要的标题：
```javascript
title: "服务项目",
```

### 6.3 修改服务项目

找到 `items` 数组：
```javascript
items: [
    {
        icon: "📷",
        name: "个人写真",
        desc: "专业人像拍摄，展现最美的你"
    },
    // ...更多项目
],
```

**修改说明：**
- `icon`：图标，可以用emoji表情
- `name`：服务名称
- `desc`：服务描述

**添加新项目：**
```javascript
items: [
    {
        icon: "💄",
        name: "新娘跟妆",
        desc: "婚礼当天全程跟妆服务"
    },
    {
        icon: "✨",
        name: "晚宴妆造",
        desc: "精致晚宴妆容设计"  
    },
    // ...可以继续添加
],
```

### 6.4 修改服务理念

找到：
```javascript
philosophy: {
    title: "服务理念",
    content: "每一次按下快门..."
},
```

修改 `content` 里的文字即可。

### 6.5 修改价目表

找到 `pricing` 部分：
```javascript
pricing: {
    title: "价目参考",
    items: [
        { name: "个人写真", price: "￥899起" },
        { name: "情侣/闺蜜", price: "￥1299起" },
    ],
    note: "* 具体价格根据..."
},
```

**修改说明：**
- `title`：价目表标题
- `items`：价格项目列表
  - `name`：项目名称
  - `price`：价格
- `note`：价格说明备注

**示例：**
```javascript
pricing: {
    title: "服务报价",
    items: [
        { name: "新娘试妆", price: "￥300" },
        { name: "新娘跟妆", price: "￥1888起" },
        { name: "妈妈妆", price: "￥388" },
        { name: "伴娘妆", price: "￥288/人" },
    ],
    note: "* 价格根据婚礼日期、地点等因素可能有所调整"
},
```

## 九、修改主题颜色

### 9.1 找到位置

在配置区找到 `theme` 部分：
```javascript
theme: {
    primaryColor: "#1a1a1a",
    secondaryColor: "#666666",
    bgColor: "#ffffff",
    cardBgColor: "#f8f8f8",
    borderColor: "#e0e0e0"
},
```

### 9.2 颜色代码说明

颜色使用 **十六进制代码**，格式为 `#RRGGBB`：
- `#000000` = 纯黑
- `#FFFFFF` = 纯白
- `#FF0000` = 红色
- `#00FF00` = 绿色
- `#0000FF` = 蓝色

### 9.3 推荐配色方案

**方案1：经典黑白（默认）**
```javascript
theme: {
    primaryColor: "#1a1a1a",    // 主色：深灰黑
    secondaryColor: "#666666",  // 次要：中灰
    bgColor: "#ffffff",          // 背景：纯白
    cardBgColor: "#f8f8f8",      // 卡片：浅灰
    borderColor: "#e0e0e0"       // 边框：淡灰
},
```

**方案2：温暖棕色**
```javascript
theme: {
    primaryColor: "#5D4037",    // 主色：深棕
    secondaryColor: "#8D6E63",  // 次要：浅棕
    bgColor: "#FFF8E1",          // 背景：米黄
    cardBgColor: "#FFECB3",      // 卡片：浅黄
    borderColor: "#D7CCC8"       // 边框：淡棕
},
```

**方案3：优雅蓝色**
```javascript
theme: {
    primaryColor: "#1565C0",    // 主色：深蓝
    secondaryColor: "#42A5F5",  // 次要：浅蓝
    bgColor: "#ffffff",          // 背景：纯白
    cardBgColor: "#E3F2FD",      // 卡片：淡蓝
    borderColor: "#BBDEFB"       // 边框：浅蓝
},
```

### 9.4 获取颜色代码

1. 打开在线取色工具：https://colorpicker.me
2. 选择你喜欢的颜色
3. 复制 HEX 代码（如 `#FF5733`）
4. 粘贴到配置中

## 十一、常见问题

### Q1：修改后页面显示空白？

**检查：**
1. 是否删除了必要的引号 `"`
2. 是否在最后一项后面加了逗号 `,`
3. 是否删除了括号 `{}` 或 `[]`

**正确格式示例：**
```javascript
name: "王小美",      // ✓ 正确
name: 王小美,        // ✗ 错误，缺少引号
name: "王小美"       // ✗ 错误，如果后面还有内容，需要逗号
```

### Q2：图片显示不出来？

**检查：**
1. 图片链接是否完整（以 http:// 或 https:// 开头）
2. 图片链接是否能在浏览器直接打开
3. 图片链接是否包含中文（建议改为英文）

### Q3：修改颜色后没变化？

**检查：**
1. 颜色代码格式是否正确（6位十六进制，如 `#FF5733`）
2. 是否保存了文件
3. 浏览器是否缓存了旧版本（按 `Ctrl+F5` 强制刷新）

### Q4：如何恢复默认设置？

保留原始的 `index.html` 文件备份，需要恢复时替换即可。

### Q5：可以修改页面布局吗？

不建议零代码基础用户修改布局。如需修改，请联系懂前端开发的朋友帮忙。
